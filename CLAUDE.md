# Books - Claude Notes

## Project Overview
Curated book rankings based on Goodreads data. Static HTML page with Apple Liquid Glass design.

## Stack
- Static HTML/CSS/JS (no build step)
- Apple Liquid Glass style (backdrop-filter, -apple-system font, #0071e3)
- Deployed to GitHub Pages via nulljosh.github.io

## Files
- `index.html` - Interactive rankings UI
- `book_rankings.md` - Markdown rankings list
- `reading-tracker.py` - Goodreads data scraper

## Development
```bash
open index.html   # Local preview
```

## Style Notes
- Apple Liquid Glass: `backdrop-filter: blur(20px)`, frosted glass panels
- Font: `-apple-system, BlinkMacSystemFont`
- Accent: `#0071e3`
